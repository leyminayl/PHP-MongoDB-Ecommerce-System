### **🛒 E-Commerce Product Management System**
A web-based application developed for the efficient management of products within an e-commerce setting. This project utilizes PHP for backend logic and MongoDB for flexible, non-relational data storage.

### 🚀 Key Features:
- **Full CRUD Operations:** Enables administrators to seamlessly Add, Edit, View, and Delete product data.
- **Role-Based Access Control:** Features distinct interfaces for the Admin Dashboard and User Interface to ensure secure access.
- **User Authentication:** Integrated secure Login and Logout functionalities to protect system data and restrict access.
- **Dynamic Dashboard:** Designed a tabular admin dashboard for efficient real-time monitoring and control of product entries.

### 🛠️Tech Stack:
- Frontend: HTML
- Backend: PHP
- Database: MongoDB Tools: Visual Studio Code, MongoDB

### 📁 Project Structure:
The system is built with structured backend logic to manage database connections and dynamic data handling.

- `admin/` – Contains the dashboard and product management control logic.
- `user/`  – Handles the client-side interface for product viewing.
-  `config/` _ – Manages MongoDB connection strings and system configurations.

### 💡 What I Learned

- Through the development of this project, I strengthened my skills in:
- Utilizing NoSQL databases (MongoDB) for handling and scaling dynamic product data.
- Implementing secure authentication flows and role-based logic using PHP.
- Designing functional UI/UX and tabular layouts for administrative data control.
